# EvoZen

基于开源增量进化游戏 [Evolve](https://github.com/pmotschmann/Evolve) 进行**现代化重构**的项目。

原版 Evolve 是一款优秀的单机网页增量游戏，玩家从原始汤中的原生质出发，经历 RNA → DNA → 物种选择 → 原始部落 → 科技发展 → 太空探索的完整进化链。EvoZen 在保留核心玩法的基础上，使用现代前端技术栈重新构建，并计划加入**多人在线**内容，让进化不再孤独。

当前仓库仍以**单机重构与核心玩法复刻**为主，多人在线部分尚处于后续规划阶段。

## 与原版的区别

| | Evolve（原版） | EvoZen |
|------|------|------|
| 技术栈 | 原生 JS + jQuery | Vue 3 + TypeScript + Vite |
| 架构 | 单文件 + 全局变量 | Monorepo + Pinia 状态管理 |
| 游戏逻辑 | 与 UI 耦合 | 纯函数引擎，零 UI 依赖 |
| 语言 | 英文为主 | 中文优先 |
| 联机 | 无 | 计划支持多人在线 |
| 存档 | 仅本地 | 本地 + 计划云端同步 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 开发检查
npm run check
```

## 项目结构

```text
EvoZen/
├── apps/
│   └── web/                  # Vue 3 + Vite 前端
├── packages/
│   ├── shared-types/         # TypeScript 共享类型定义
│   └── game-core/            # 纯游戏逻辑引擎（零 UI 依赖）
├── scripts/                  # 开发工具脚本（审计 / 校验）
├── parity/                   # 与原版对齐进度追踪（manifest.json）
├── legacy/                   # 原版源码归档（仅参考，不参与构建）
└── package.json              # npm workspaces 根配置
```

## 开发路线

### 第一阶段 A：文明时代核心

用现代技术栈复刻核心玩法，跑通完整的文明时代单机循环。

- 已完成：基础资源/建筑/科技/岗位流转、军事与贸易、电力网格、基础种族等文明时代核心循环（目前 MVP 仅使用 humanoid 路线进行打通测试）。
- 已完成：银行 / 赌场 / 旅游 / ARPA 基础（纪念碑、证券交易所）等全球化尾项。
- 待补齐：宗教体系、地形与天气、完整的 50+ 进化分支与所有 30+ 种族。

### 第一阶段 B：发现时代 / 工业化

在文明时代核心完成后，推进至工业化和全球化。

- [x] 间谍/外交系统完善
- [ ] ARPA 进阶项目（超级对撞机/基因组/纳米机器人）
- [ ] 发现时代剩余科技与制度分支
- [ ] 扩充种族路线（植物/真菌/昆虫等 30+ 种族）

### 第一阶段 C：太空时代（当前）

还原原版完整的太空探索阶段。

- 已完成：spc_home 4 座（卫星 / 推进剂储备站 / GPS / 导航信标）+ spc_moon 3 座（月面基地 / 铱矿 / 氦-3 采集站）；统一的支援池 / 跨区支援 / 燃料裁剪抽象已就绪。

- 待补齐：进化树扩充（植物/真菌/合成等非 humanoid 路线）＋全 30+ 种族
- 待补齐：小行星采矿、气态巨星

### 第一阶段 D：深空 / 星际 / 传送门

- 深空探索与星际旅行
- 传送门/地狱系统
- 最终阶段（伊甸/真实路径）

### 第二阶段：后端服务

搭建服务端，实现云端存档和反作弊。

核心目标：让玩家在同一个世界中共同进化、竞争与合作。

- 多文明并存，外交/贸易/战争
- 排行榜与赛季机制
- 公会/联盟系统

## 设计原则

1. **绝对 1:1 数值复刻** — 单机核心逻辑、数值平衡与限制公式绝对且严格对准 Legacy 原版代码，不做主观魔改。
2. **逻辑与 UI 分离** — `game-core` 是纯函数引擎，为未来的云端或联机复用留作基础。
3. **渐进式开发** — 采用敏捷开发打通最小可用分支（MVP），再平铺扩充原版的庞大系统。
4. **中文优先** — 以重构中文版体验为优先，暂不做国际化。
5. **legacy 只读** — 旧代码作为绝对的数据字典，仅供查阅，完成功能覆盖后将移除。
